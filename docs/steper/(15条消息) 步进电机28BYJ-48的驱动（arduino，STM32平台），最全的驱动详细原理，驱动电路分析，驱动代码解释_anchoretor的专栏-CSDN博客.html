一、步进电机28BYJ-48结构和工作原理

1.基本知识：

步进电机：是数字控制电机，它将脉冲信号转变成角位移，即给一个脉冲信号，步进电机就转动一个角度，因此非常适合于单片机控制。步进电机可分为反应式步进电机（简称VR）、永磁式步进电机（简称PM）和混合式步进电机（简称HB），后两种常用。

步进电机控制特点：1、它是通过输入脉冲信号来进行控制的。2、电机的总转动角度由输入脉冲数决定。3、电机的转速由脉冲信号频率决定。

三.驱动代码分析

arduino自带的steper驱动库，针对两相电机，而我们驱动的28BYJ48步进电机时四相的，直接使用不可以，笔者试用发现电机嗯嗯响不转动，如果要用自带库，可以对电机接线进行调换改造，或者修改驱动里的拍子表，太麻烦一些。

我们采用自建的驱动库，知道工作原理，驱动库可以自建。

  1. // library interface description

（2）应用案例

应用这个库编写了案例，实现功能，通过串口1进行交互，输入转动角度，实现控制电机转动角度，修改代码可以体验驱动步进电机拍数3种模式，1:EightBeat，2:FourBeat，3:DoubleFourBeat

代码使用说明在程序头部。

  1. /*接线：务必不要错了，不然查错非常头大！
  2.   L298输入1-4 对应 arduino数字端口 8,9,10,11
  3.   L298电路板输出有两种接口，一种是有步进电机的卡扣端子ABCD+vcc，一种是输出端子：out1--A+白,OUT2--A-蓝，out3--B+黄，out4--B-绿，电源端接红线5V
  4.   或者 红线接电源5V，橙色电线接out4口，黄色电线接out3口，粉色电线接out2口，蓝色接out1口根据提示进行接线。
  5.   //思路是通过减速比来控制单步执行的脉冲次数，电机是减速比16,电机每一个脉冲前进5.625度,而主轴前进5.625/16=0.3515625度,但经过我测试,这种玩具级的电机主轴低于5.625度步进的精度是无法保障的,因此我简化了代码,每步执行16次脉冲。
  6. //接线顺序是红->﹢5V,橙黄粉蓝分别接L298N 的OUT1、OUT2、OUT3、OUT4, arduino的8,9,10,11分别接L298N的IN1-IN4
  7. */
  8. /*28BYJ-48步进电机的齿轮减速比常见有两种，1：16，1：64，
  9.  * 步距角：θ=360度/（转子齿数J*运行拍数），齿数为8，八拍运行，电机主轴的步距角为θ=360度/（8*8）=5.625度（俗称半步），在除以减速比64，所以输出轴的步距角：5.625/64。
  10. 电机参数表中的减速比这个参数吧——1:64，转子转64圈，最终输出轴才会转一圈，也就是需要64*64=4096个节拍输出轴才转过一圈，2 ms*4096=8192 ms，8秒多才转一圈4096个节拍转动一圈，那么一个节拍转动的角度——步进角度就是360/4096，看一下表中的步进角度参数5.625/64。
  11. 要输出某一角度degree，求输出步数N=(degree*减速比)*（齿数*运行拍数）/360
  12. 启动频率是≥550，单位是 P.P.S，即每秒脉冲数，这里的意思就是说：电机保证在你每秒给出550个步进脉冲的情况下，可以正常启动。那么换算成单节拍持续时间就是 1 s/550=1.8 ms，为了让电机能够启动，我们控制节拍刷新时间大于 1.8 ms 就可以了。
  13. 减速比为1：64情况下
  14.   4步控制信号序列：11.25度/步，主电机32步旋转一周。
  15.   8步控制信号序列：5.625度/步，主电机64步旋转一周，减速轴旋转一周将用：64（步/周）X64（齿轮比） = 4096 步
  16.   测试电机减速比为1：16，8拍模式下旋转一周将用：64（步/周）X16（齿轮比） = 1024 步。*/
  17. #include "Steper_28BYJ48.h" //自带驱动，也可以加入驱动库
  18.  
  19. Steper_28BYJ48 steper(16, 1, 8, 9, 10, 11);
  20. //6个参数为：第1个是步进电机减速比，第2个是步进电机拍数模式设定，1:EightBeat，2:FourBeat，3:DoubleFourBeat，另外4个驱动引脚
  21. int degree;                  //转动的角度，负值为逆时针方向。
  22. String inString = "";    // 用于转换数字的字符串缓冲区
  23. String read_String = "";
  24. unsigned char dir = 3; //1顺时针方向。clockwise,2逆时针。3，空闲等待。
  25.  
  26. void setup() {
  27.   Serial.begin(9600);  //初始化串口，作为监视。
  28.   steper.setSpeed(3);//节拍刷新时间，一般要大于2ms// delay between steps, in ms, based on speed,if delay<2ms,lost step.so delay must be more than 3ms
  29.   while (!Serial) {
  30.     ; // wait for serial port to connect. Needed for Leonardo only
  31.   }
  32.   // 发送信息
  33.   Serial.println("Please input MOTOR diretion and turn degree(eg,-24,mean,counterclockwise,24 du)");
  34. }
  35.  
  36. void loop() {
  37.   int inChar;
  38.   // 读取串口发送的信息:
  39.   while (Serial.available() > 0)
  40.   {
  41.     inChar = Serial.read();
  42.     read_String += (char)inChar;
  43.     delay(2);
  44.   }
  45.   if (read_String.length() != 0) {
  46.      Serial.println("**********************************************************************");
  47.     Serial.print("serial port read_string:");
  48.     Serial.println(read_String);
  49.     if (read_String.charAt(0) == '-')
  50.     {
  51.       dir = 0;
  52.       Serial.print("turn direction:");
  53.       Serial.println("CounterClockWise");
  54.       degree = (read_String.substring(1)).toInt();
  55.     }
  56.     else
  57.     {
  58.       Serial.print("turn direction:");
  59.       Serial.println("ClockWise");
  60.         dir = 1;
  61.         degree = read_String.toInt();
  62.    
  63.     }
  64.     read_String = "";
  65.   }
  66.   if (dir == 1)//CW
  67.   {
  68.     steper.step(degree);
  69.     steper.lowPower_idel();
  70.         Serial.print("Motor finished degree: ");
  71.         Serial.println(degree);
  72.        Serial.println("**********************************************************************");
  73.  
  74.      dir = 3;   
  75.   }
  76.    if (dir == 0)//CCW
  77.   {
  78.     degree=0-degree;
  79.     steper.step(degree);
  80.     steper.lowPower_idel();
  81.     dir = 3;
  82.         Serial.print("Motor finished degree: ");
  83.         Serial.println(degree);
  84.         Serial.println("**********************************************************************");
  85.  
  86.     }
  87.     degree = 0;
  88.     delay(5000);
  89.  }

总结

1首先需要熟悉步进电机28BYJ-48的机构，工作原理。驱动拍的工作特点。关键：五线四相结构，驱动的步进角公式会计算。

2..驱动库编写需要针对不同驱动电路的设计，驱动电路的使用工作原理。关键：ULn2003驱动电路特点，输入和输出是反相，输入高电平，驱动输出端拉低，驱动绕组励磁工作。L298N驱动电路特点，输入和输出是同相，输入低电平，驱动输出端拉低，驱动绕组励磁工作。

3.应用代码分析，通过应用大家熟悉如何使用自建库，同时完成实践应用，知行合一多多练习。
